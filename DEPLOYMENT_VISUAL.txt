WHAT IF SOMETHING BREAKS:
=========================

Problem: "Connection failed"
→ Check config.php has correct credentials
→ Verify database exists on live server

Problem: "Page not found" (404)
→ Check files uploaded to public_html/fkpark/
→ Check folder structure matches local

Problem: "Login fails"
→ Check users table imported correctly
→ Try admin / admin

Problem: "QR code doesn't scan"
→ Make sure domain is accessible from internet
→ Test by typing URL manually in browser
