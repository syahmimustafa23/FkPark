🎯 QUICK START (DO THIS NOW):
========================================

The ONLY 4 values you need to change in config.php:

BEFORE (Your Local Setup - Right Now):
──────────────────────────────────────
$db_host = "localhost";
$db_user = "root";
$db_pass = "";
$db_name = "fkpark";

AFTER (When You Deploy to Live Server):
────────────────────────────────────────
Get these 4 values from your hosting email/cPanel:

$db_host = "YOUR_DATABASE_HOST";
$db_user = "YOUR_DATABASE_USER";
$db_pass = "YOUR_DATABASE_PASSWORD";
$db_name = "YOUR_DATABASE_NAME";

That's literally it!

Example from Hostinger:
$db_host = "mysql.hostinger.com";
$db_user = "u123456_user";
$db_pass = "SecurePass123";
$db_name = "u123456_fkpark";

========================================
📋 3-STEP DEPLOYMENT PROCESS:
========================================

STEP 1: Copy Your Database
──────────────────────────
Local (XAMPP):
  → Go to http://localhost/phpmyadmin
  → Select database "fkpark"
  → Click "Export"
  → Save as fkpark_backup.sql

Live Server (Hosting):
  → Login to cPanel
  → Go to phpMyAdmin
  → Select your live database
  → Click "Import"
  → Upload the file you saved

STEP 2: Update config.php
──────────────────────────
Open: c:\xampp\htdocs\FkPark\config.php
Edit: 4 lines (the values above)
Save: Ctrl+S
Upload: Via FTP to server

STEP 3: Upload All Files
────────────────────────
Use: FileZilla (free FTP software)
From: c:\xampp\htdocs\FkPark\
To: public_html/fkpark/
Wait: For upload to complete
Test: Open https://parkingapp.com/fkpark/login.php

Done! ✅ App is live!

🎓 KEY CONCEPTS EXPLAINED:
========================================

HOSTING:
→ A computer on the internet running your app
→ 24/7, always online
→ Costs $3-10/month
→ Handles domain name, email, database

CONFIG-BASED:
→ One file (config.php) tells your app where database is
→ Move servers = just change that one file
→ No code changes needed
→ Very portable and flexible

DATABASE MIGRATION:
→ Export data from local computer
→ Import data to live server
→ Everything copied (users, bookings, vehicles)
→ Data is identical on both

FTP:
→ File Transfer Protocol
→ Way to upload files to server
→ Like copying to USB, but online
→ Use FileZilla (free software)

QR CODES:
→ Automatically use your domain/IP
→ No hardcoding needed
→ Works on localhost and live server
→ Scans from any phone camera

========================================
💡 INSIDER TIPS:
========================================

TIP 1: Keep backup of original config.php
└─ Save current version before deployment
└─ You can switch back if needed

TIP 2: Test locally first
└─ Make sure everything works on XAMPP
└─ Then deploy with confidence

TIP 3: Use HTTPS on live
└─ Most hosting includes free SSL
└─ Secure passwords, secure data
└─ QR codes work with HTTPS

TIP 4: Change default password
└─ After deployment, change admin password
└─ Don't use "admin/admin" forever

TIP 5: Monitor logs
└─ Check hosting error logs if something breaks
└─ Usually in cPanel → Error Logs
└─ Helps debug issues
